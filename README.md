tap
===

Hotel management analytics panel

### Configuring `app.yaml`

The default contents of `app.yaml` is as follows:

```
default:
  server:
    port: 7000
    host: '127.0.0.1' 
test:

production:
  server:
    port: 8000
    host: '127.0.0.1' 
```
